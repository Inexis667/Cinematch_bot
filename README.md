# CineMatch — Умный кино-бот 🎬

Telegram-бот для поиска фильмов по настроению, создания коллекций и получения AI-рекомендаций.

## Функциональность
- Поиск фильмов с фильтрацией
- Подборки по настроению
- Личные коллекции фильмов
- Кино-рулетка
- AI-рекомендации

## Технологии
- Python 3.10+
- Aiogram 3.x
- PostgreSQL
- Redis
- TMDb API / Kinopoisk API

## Установка
1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости
4. Настроить .env файл
5. Запустить бота
